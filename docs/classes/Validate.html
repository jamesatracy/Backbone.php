\Validate

Valdiation class for testing values against validation rules.

Formerly class "SchemaRules"

Built-in rules include: "required" => true: field is required (cannot be empty) "numeric" => true: field must be numeric (0-9 chars only) "email" => true: validate email address "url" => true: validate url (with path) "min" => value: require a minium value for a field "max" => value: require a maximum value for a field "minlength" => value: require a minimum length for a (string) "maxlength" => value: require a maximum length for a (string) "enum" => array(): value must in the given array "binary" => true: value is either 0 or 1

Note that all rules, except required, will validate if the value is empty. In other words, it assumes that the rule applies only if a value is present. The required rule should be used to catch cases where a value should never be empty and then combined with any of these additional rules.
