\Router

Class for creating and dispatching routes.

Examples: Router::get("/", "function_name"); Router::get("/path/", "/controllers/AppController@handlePath") Router::get("/path/", "View@path-view");

alias()

alias(string $name) : \Router

Create an alias for the route.

An alias alows you to construct a fully qualified url for a valid route, including route parameters, without the need to manually type out the path. As such, you could change the route for a path without the need to also change all references to it elsewhere.

Parameters

string $name

The name of the alias.
