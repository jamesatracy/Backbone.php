link()

link(string $subpath, bool $print, bool $ssl) : string

Builds a link from the base and the given subpath (or relative path)

Ex: link("/about/team/") => http://www.example.com/about/team/

Parameters

string $subpath

The relative path
bool $print

If true, then echo the link, otherwise return the string
bool $ssl

Force the link to SSL, if not already

Returns

string —

The link if $print = false
