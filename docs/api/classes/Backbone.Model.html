\Backbone

\BackboneModel

Base Model class for database objects.

Provides basic methods for creating, updating, fetching, and validating model data. This class can be used on its own or extended to implement more specialized model functionality.

toJSON()

toJSON(bool $compact) : array

Return a JSON representation of the model.

This is not a string, but an associative array that you can pass to json_encode().

The JSON is compacted by leaving out the field names and returning the values as a plain array.
