Classes

Validate Valdiation class for testing values against validation rules.
Response Handles sending HTTP response headers and content.
Model Base Model class for database objects.
ModelQuery Extends the Query class so executing select statements on a Model can return a Collection object.
View Base class for all Backbone.php views.
Request Convenience class for accessing the request URI, generating links, and working with $_POST, $_GET, and $_FILES.
Sanitize Class for sanitizing strings, including striping html and escpaing for database input.
Schema Schema represents the definition of a data resource and is the base class for Backbone.php's Model class.
Collection Collection class for representing an ordered list of model objects.
Events Class for triggering and delegating events and callbacks.
